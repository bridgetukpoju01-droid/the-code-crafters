## LOOPS

In Go, the `for` loop helps you repeat a block of code several times. It's the only type of loop in Go. Each time the loop runs, it's called an "iteration." The loop has three parts: setting the starting point, checking if the loop should continue, and updating the counter after each run. For example, you can use a `for` loop to print numbers from 0 to 4 by starting at 0, continuing while the number is less than 5, and increasing the number by 1 each time. You can also change how much the counter increases, like counting by tens up to 100. The `continue` statement skips an iteration, while `break` stops the loop completely. You can also have loops inside loops, called nested loops, to do more complex tasks. The `range` keyword helps loop through arrays or lists easily, giving you both the index and the value of each item. You can skip showing either the index or the value by using an underscore (`_`). This makes the `for` loop in Go very useful for repeating actions, skipping steps, or working with lists.
