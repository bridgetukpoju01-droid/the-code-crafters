## GO STATEMENT
In Go, fmt.Println("Hello World!") is a statement, and statements end when you press Enter or use a semicolon ;, with Go automatically adding a semicolon at line breaks. Curly braces {} should usually be at the end of a line, not the start, though Go allows them on the next line for func declarations. For example, this code prints “Hello World!”: package main; import "fmt"; func main() { fmt.Println("Hello World!") }. Here, package main defines the program, import "fmt" allows printing, and func main() is the starting point. Running go run main.go displays Hello World!. The key points are that Go ends statements with newlines or semicolons, adds semicolons automatically, and requires proper brace placement for clear code.
