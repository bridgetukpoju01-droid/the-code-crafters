In Go, **arrays** are used to store multiple values of the same type in a single variable, and they have a fixed size. You can declare an array in two ways: using the `var` keyword, where you either define the length and type (e.g., `var arr = [3]int{1, 2, 3}`) or let Go infer the length (e.g., `var arr = [...]int{1, 2, 3}`), or using the shorthand `:=` (e.g., `arr := [3]int{1, 2, 3}`). Once declared, the array can hold a fixed number of elements. You access and change elements of an array by referring to their index, starting at `0`. For example, `arr[0]` accesses the first element. Arrays are automatically initialized to default values, like `0` for integers or `""` for strings, but you can also initialize specific elements by specifying the index, such as `{1:10, 2:40}`. To find the length of an array, you can use the `len()` function. For example, `len(arr)` gives the number of elements in the array. Arrays in Go help store multiple related values of the same type efficiently.
