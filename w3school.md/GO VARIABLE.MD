## GO VARIABLE
n Go, a variable is like a container that holds a value. There are different types of variables: int for whole numbers, float32 for decimal numbers, string for text, and bool for true or false. You can create a variable in two ways: using the var keyword, where you can write the name, type, and value, or using :=, where Go automatically figures out the type from the value. Variables can have a value when you create them or later, and if you don’t give a value, Go assigns a default one (0 for numbers, "" for text, and false for booleans). The var keyword can be used inside or outside functions and allows declaring and assigning separately, while := can only be used inside functions and must have a value immediately. For example, inside a function you can write myNum := 50 and then use fmt.Println(myNum) to display the value 50.
